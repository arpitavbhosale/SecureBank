About This Project

Project Details

Title: Data Privacy Preservation System using Homomorphic Encryption Technique
Domain: Banking & Financial Security
Encryption: Paillier Homomorphic Encryption (512-bit)
Storage: Firebase Firestore — encrypted balances stored as ciphertext strings
Auth: Firebase Authentication (Email / Password) — each user's data is fully isolated
Technologies: Python, Flask, Firebase, HTML / CSS / JavaScript

What is Homomorphic Encryption?

Homomorphic encryption allows computations to be performed on encrypted data without decrypting it first. Banking data remains encrypted throughout all operations — deposits, withdrawals, transfers, and balance aggregation — while still producing correct results.

Key Features

  • ✅ Paillier homomorphic encryption (512-bit) — implemented from scratch
  • ✅ Encrypted balances stored in Firebase Firestore as ciphertext strings
  • ✅ Firebase Authentication — each user has private, isolated data
  • ✅ Every API request verified with a short-lived Firebase ID token
  • ✅ Privacy-preserving total balance calculated directly on ciphertext
